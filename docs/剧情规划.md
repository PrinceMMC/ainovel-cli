# 剧情规划

## 大纲概述
全书分5卷，主角从觉醒→猎艳→布局→巅峰→收尾。

### 时间线与章节规划
**卷一：觉醒与初尝（1-50章）**
- 开篇：林辰被甩，买飞机杯意外激活苏婉儿体液（偷自她遗留物品）。
- 首次通感：远程NTR苏婉儿，感受到征服快感。
- 支线：接近白薇，偷取体液。

**卷二：公司与校园（51-150章）**
- NTR白薇、李梦瑶等多女。
- 升级飞机杯，解锁记忆读取。
- 冲突：陈浩察觉异常但无证据。

**卷三：黑市与豪门（151-300章）**
- 进入欲界，交易更强淫具。
- 大型NTR事件：同时通感多名女性，举办“隐秘派对”。
- 伏笔：欲界组织邀请主角加入/对抗。

**卷四：权力扩张（301-500章）**
- 利用通感操控商业/政治。
- 后宫建立，女性们逐渐沉沦依赖主角。
- 高潮冲突：多角NTR大战。

**卷五：终局（501+章）**
- 揭秘通感起源，反派BOSS战。
- 主角建立欲望帝国，完美收官。

## 主要伏笔
- 飞机杯真正来历与副作用。
- 女性们逐渐意识到“梦中情人”是林辰。
- 绿帽男的反击与失败。

## 爽点设计
每10-20章一次大型肉戏，详细描写通感过程、女性反应、NTR羞辱。
