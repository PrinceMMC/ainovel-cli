# 写作规则

## 文风
- **极致细腻肉戏**：使用大量感官描写（视觉、触觉、声音、气味），术语露骨（鸡巴、骚穴、淫水、子宫等）。
- **第一人称或第三人称有限视角**：主角视角为主，强调心理活动与征服快感。
- **节奏**：日常推进+突发肉戏+阴谋布局。肉戏占40%，剧情占60%。

## 创作规范
- **NTR强调**：突出“别人老婆/女友在主角飞机杯里浪叫”的背德感，绿帽男对比弱鸡。
- **人物刻画**：女性从抗拒/羞耻到沉沦，层层递进。主角逐步黑化但不崩人设。
- **禁止**：无脑后宫无逻辑；避免纯虐；确保每场通感有新意（不同体位、场景、升级效果）。
- **章节结构**：每章2000-4000字，开头铺垫、中间冲突、高潮肉戏、结尾埋伏笔。
- **语言**：中文网络爽文风格，口语化+文雅结合，色情描写生动不重复。

**更新建议**：每日1-2章，读者反馈调整支线。
